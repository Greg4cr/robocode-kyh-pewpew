Welcome to the robocode-kyh-pewpew project page

robocode-kyh-pewpew battling Robocode is an open source game project that is available in the programming languages of Java and the .NET framework. The objective of the game is to be able to create a robot that will defeat all the other enemy robots on the playing field.

For this project, we have created a competitive robocode robot called PewPew.
